# Wood Cutting Optimizer

A material nesting optimization tool designed for efficient cutting of wood panels and beams with waste minimization.

![Screenshot Placeholder](https://via.placeholder.com/800x400)

## Features

- **Dual Material Support** - Optimize cutting for both Platen/Sheet (plywood, MDF) and Balken/Beam (hardwood, lumber) materials
- **Predefined Stock Sizes** - Built-in library of standard plywood, MDF, and hardwood dimensions
- **Part Library** - Common part sizes for quick selection
- **Custom Dimensions** - Define custom part sizes and material stocks
- **Visual Layout Preview** - See 2D cutting layout before production
- **Waste Calculation** - Automatic computation of material waste percentage
- **Cost Estimation** - Calculate total project cost based on material and part count
- **Import/Export** - Save and load cutting plans for future projects
- **Optimization Algorithm** - Intelligent packing to minimize scrap material

## Tech Stack

- HTML5
- JavaScript
- Canvas API

## Getting Started

Access the optimizer online:

1. Visit: [Wood Cutting Optimizer](https://arnoutzw.github.io/woodcut-optimizer/wood-cutting-optimizer.html)
2. Select material type (Sheet or Beam)
3. Choose or define stock sizes
4. Add parts needed for your project
5. Run optimization
6. Review layout and waste calculations
7. Export cutting plan

For local development:
```bash
git clone https://github.com/arnoutzw/woodcut-optimizer.git
# Open wood-cutting-optimizer.html in your browser
```

## Usage Workflow

1. **Define Stock Material** - Select predefined sizes or add custom dimensions
2. **Add Parts** - Use part library or specify custom dimensions
3. **Set Optimization Goals** - Minimize waste, minimize cost, or balance both
4. **Run Optimizer** - Generate cutting layout
5. **Review Results** - Check waste percentage and estimated cost
6. **Export Plan** - Save as project for reference or machine control

## Material Types

- **Platen/Sheet**: Plywood, MDF, hardboard
- **Balken/Beam**: Lumber, hardwood beams, structural timber

## Benefits

- Reduce material waste 15-30%
- Lower project costs
- Optimize cutting sequences
- Reuse scrap for secondary projects
- Professional project documentation
